- 회원가입
- 로그인
  
- Todo 입력
- Todo 수정
- Todo 삭제
- Todo 완료

- 음악 재생

### Prerequisites

**Node.js 18.17.0**

### Cloning the repository

```shell
git clone https://github.com/sossost/todolist.git
```

### Install packages

```shell
npm i
```


### Start the app

```shell
npm start
```
