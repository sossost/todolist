### Key Features:

- Sign up
- Sign in
  
- Todo create
- Todo modify
- Todo delete
- Todo complete

- Music playing

### Prerequisites

**Node.js 18.17.0**

### Cloning the repository

```shell
git clone https://github.com/sossost/todolist.git
```

### Install packages

```shell
npm i
```


### Start the app

```shell
npm start
```
